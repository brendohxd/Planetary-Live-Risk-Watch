# Earth Risk Watch · Australia

This is the **Australia branch** — the localised build.

| Branch | Purpose |
|--------|---------|
| `main` | Clean **baseline template** (no country baked in) — fork from here for other countries |
| `australia` | **This branch** — Australia-first plain language, BOM, local heat/fire/El Niño context |

Point **GitHub Pages** at the `australia` branch so the live AU site stays Australian while `main` stays a clean template.

## Official warnings always win

This is **not** an official warning system.

- [Bureau of Meteorology](https://www.bom.gov.au/)
- Your **state emergency service** / fire service apps
- Local council and police directions in an emergency

## Design rules

1. **Local first** — present + historic Australian facts people can verify
2. **Plain language** — if something is serious, say **serious**
3. **Truth-seeking** — prefer documented agency numbers (e.g. NOAA/CPC El Niño probabilities) even when soft headlines lag
4. **Official warnings win** — BOM and state services beat this site

## Fork for another country?

Start from **`main`**, not this branch, so you do not have to strip Australian agencies, wording, or examples.

See `LOCALIZE.md` on `main`.
